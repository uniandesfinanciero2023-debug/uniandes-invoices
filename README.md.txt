# Sistema de Revisión de Facturas UNIANDES

Sistema web para la revisión de cuentas por pagar a proveedores de la Universidad Regional Autónoma de los Andes.

## Características

- Gestión de facturas de proveedores
- Verificación de documentos (facturas, retenciones, comprobantes)
- Checklist de validación
- Interfaz moderna y responsive

## Cómo usar

1. Sube estos archivos a GitHub Pages
2. Accede a tu página web
3. Usa el formulario de búsqueda para encontrar facturas
4. Completa el proceso de revisión

## Estructura

- `index.html` - Aplicación web completa
- `db.json` - Datos de ejemplo (simula una base de datos)
